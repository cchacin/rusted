# rusted

This repo contains some fun code in Rust. Install rust and Cargo and get going

## Rust Installation

### From Homebrew

```bash
$ brew install rust
```

### From sh.rustup.rs

```bash
$ curl https://sh.rustup.rs -sSf | sh
```

## Building and Running using cargo

Enter to the appropiated directory, i.e:

```bash
$ cd christmas_tree
```

Build and run:

```bash
$ cargo build && cargo run
```
